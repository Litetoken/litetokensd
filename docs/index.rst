litetokensd
==================================================

Litetokens is a protocol for the creation and use of decentralised financial instruments such as asset
exchanges, contracts for difference and dividend payments. It uses Litecoin as a transport layer.
The Litetokens protocol specification may be found `here <https://github.com/LitetokensXLT/Litetokens>`__.

``litetokensd`` is the reference client (and server daemon) implementation of the Litetokens protocol.

To get ``litetokensd`` installed and set up on your computer, you have two options:

- Set it up manually, using the instructions `here <https://github.com/LitetokensXLT/litetokensd/blob/master/README.md>`__
- For Windows and Ubuntu Linux users, you can use the `automated build system <http://litetokens.io/docs/build-system/>`__
